# MarioEvolution

## Build

Toàn bộ game và tool dùng Makefile tại thư mục gốc:

```powershell
make create
make run
make map-editor
make run-map-editor
make render-preview
make run-render-preview
make test
make clean
```

## Map

Game và map editor dùng chung `assets/level1.map`:

```text
B  StandardBrick
?  CoinBrick
M  MushroomBrick
F  FlowerBrick
o  Coin tự do
.  Ô trống
```
